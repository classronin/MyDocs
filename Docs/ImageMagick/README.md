# im.py - 图片格式转换与处理工具

基于 ImageMagick 的命令行图片处理脚本，支持中文指令、尺寸调整和批量操作。

## 功能特性

- 格式转换：支持 PNG、JPG、WebP、AVIF 等常见格式互转。
- 旋转翻转：右旋、左旋、左右翻转、上下翻转。
- 智能缩放：支持多种尺寸写法。
- 批量处理：一键处理当前目录所有图片，自动创建输出文件夹。
- 防覆盖：文件名冲突时自动添加序号。

## 环境依赖

- [ImageMagick](https://imagemagick.org/) （需将 `magick` 或 `convert` 加入系统 PATH）
- Python 3.8+（建议使用 `uv` 管理项目环境）

## 安装与配置

1. 安装 ImageMagick，确保命令行可执行 `magick` 或 `convert`。
2. 将脚本目录加入系统 PATH，或直接使用配套的 `.bat` 文件。

## 使用方法

### 基本语法
```
im.py <目标格式> [动作] [尺寸] <文件...>
```


### 动作指令
- 旋转：`右旋` / `右`（顺时针 90°）、`左旋` / `左`（逆时针 90°）
- 翻转：`左右翻转` / `左右`、`上下翻转` / `上下`
- 批量：`批量` / `批`

### 尺寸指令

以下写法均可识别，内部自动转换为 ImageMagick 的 `-resize` 参数。

- 纯数字：`500` → 宽度 500，高度等比缩放。
- 宽度缩写：`500x` → 同上。
- 高度缩写：`x500` → 高度 500，宽度等比缩放。
- 固定盒子：`500x800` → 将图片适配进 500×800 范围内，保持比例。
- 中文写法：`宽500`、`宽度500`、`高500`、`高度500`。

多个动作或尺寸参数同时出现时，以最后一个为准。内部执行顺序固定为“先旋转/翻转，后缩放”。

### 单文件处理示例

```bash
# 转换为 PNG，右旋 90°
python im.py png 右旋 photo.jpg

# 转换为 WebP，宽度缩放至 500
python im.py webp 500 photo.png

# 转换为 JPG，高度缩放至 300
python im.py jpg x300 photo.bmp

# 转换为 AVIF，限制在 800x600 范围内
python im.py avif 800x600 photo.tiff
```

批量处理示例
```
# 将当前目录所有图片转为 PNG（输出至 out-0 文件夹）
python im.py png 批量

# 批量转为 WebP，并统一宽度 800
python im.py webp 批量 800

# 批量转为 JPG，先右旋再限制尺寸
python im.py jpg 批量 右旋 500x500
```

通过 .bat 快捷调用
若已配置 .bat 文件（如 PNG.bat、WebP.bat），可直接在命令行使用：
```
PNG 右旋 500 photo.jpg
WebP 批量 800
```

注意事项
>批量模式只处理当前目录下的图片文件，不递归子文件夹。
>若输出文件已存在，会自动生成 文件名-0.扩展名 等序号文件。
>尺寸参数中的 500x800 为最大边界限制，输出图片不会超过该尺寸，但会保持原始宽高比。
