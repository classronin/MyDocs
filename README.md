[Aria2c](https://github.com/aria2/aria2)批量下载至CMD当前目录的路径
```
ba https://.../10*.jpg   # 请输入网址模板 (使用 * 代替数字部分，如 https://.../10*.jpg)
01-21                    # 请输入数字范围 (格式如 01-21 )
```

[Surge](https://github.com/surge-downloader/Surge)批量下载至CMD当前目录的路径 
```
bs https://.../10*.jpg   # 请输入网址模板 (使用 * 代替数字部分，如 https://.../10*.jpg)
01-21                    # 请输入数字范围 (格式如 01-21 )
```

[Surge](https://github.com/surge-downloader/Surge)单文件下载，文件下载至CMD当前目录的路径
```
s https://...zip   # 下载至CMD当前目录的路径
```
