Sapp.py 使用说明

功能：
- 自动搜索 Scoop 所有已安装 bucket 中的应用（无需手动指定 bucket 列表）
- 使用 surge 加速下载 GitHub 资源
- 自动将文件保存到 Scoop 缓存目录

使用方法：
    sapp <应用名称>
例如：sapp uv

说明：
- 支持应用名称的部分匹配，如输入 ollama 会匹配 ollama 和 ollama-full
- 自动遍历 %SCOOP%\buckets 下的所有 bucket 目录
- 优先使用 64 位架构的下载链接
- 缓存文件命名规则：应用名#版本#URL哈希值.扩展名

注意：
- 确保已安装 surge 工具
- 确保已添加所需的 bucket 并运行过 scoop update
